# ProCropper PDF

一个基于 Flutter 的跨平台 PDF 裁边工具，致力于在移动端与桌面端提供更智能、更高效的 PDF 裁边体验，无需依赖额外的运行环境。

本项目参考了Briss的叠加预览、分组裁切思路，并在此之上进行了优化和改进，能够更智能地根据内容页墨迹排布的不同，划分出不同的分组，并自动生成裁切区域，减少用户的操作步骤。

# 功能特性

- [x] PDF 叠加预览
- [x] 页面智能分组+手动编辑
- [x] 自动生成裁切区域
- [x] 手动过滤页面边缘区域
- [x] 锁定纵横比
- [x] 向外留白模式
- [x] Android和iOS的分享导入
- [ ] 处理加密的 PDF
- [ ] 应用内预览 PDF
- [ ] 多实例
